# reports/

Session reports for the We Roll Spinners / Future Spinner project.

- `SESSION_REPORT.md`: the latest session's report. Overwritten each session with a
  summary of what ran, what changed, verification results, and anything needing owner
  attention (per the Session Conventions in `CLAUDE.md`).
- `archive/<date>_<topic>.md`: a dated copy of each session report, kept for history.

Convention: at the end of every session, write `SESSION_REPORT.md`, copy it to
`archive/<date>_<topic>.md`, then commit and push both.
